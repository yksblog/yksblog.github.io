vpn卡，网速慢

　　想要科学上网，于是买了海外的VPS。通过代理可以实现科学上网。奈何vpn卡，网速慢，网页打开龟速一样。体验实在难受。而一般VPS只是装了命令行界面，如果命令行界面可以网上冲浪就好了。于是，就有了这篇文章：使用emacs的eww浏览器，在命令行上进行海外网上冲浪。

安装emacs28-nox

$ sudo apt remove --autoremove emacs
$ sudo add-apt-repository ppa:kelleyk/emacs
$ sudo apt install update
# （以用于纯文本界面）
$ sudo apt install emacs28-nox

emacs配置

　　拉取我自己的emacs配置，就省略了。

设置alias

　　既然用了emacs，那必须得上emacsclient啊。 

# emacs
alias ess="emacs --daemon"
# -t 终端；-c GUI
alias e="emacsclient -t "
alias ekk="kill -9 `ps aux | grep emacs | grep -v grep | awk '{print $2}' `"
alias eps="ps aux | grep emacs | grep -v grep"

　　好了，好了终于可以在命令行，海外网上冲浪了。
