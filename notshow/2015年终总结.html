　　2014年下学期，我大四了，本着学校提倡的：就业优先，的准则，我和小伙伴离开学校，然后，去了从小心心念念的首都——北京。这一年，换了四份工作，换了两个城市。

北京葡萄科技:比特币交易平台(年后到3月15号离职）

　　SpringMVC + mysql + myBaties + redis 缓存。这家公司的技术框架是「魏工」从上个公司的带过来的。是当前普遍的技术。核心是交易引擎，但是一般开发技术不到。

优点:

  • 包吃包住月薪6500 (税后）
  • 同事相处还算融洽(赵亚杰，葛兴华，卢霆锋，吴建飞，苏伟)

对于一个刚开始工作的程序员来说，有许多实践的机会。我承认我的技术的这家公司得到了很大提升。

缺点:

  • 作六休一
  • 无明确的时间概念。
  • 整天吃住都在公司，除了吃睡，就是工作。薪资待遇不高，后来勾心斗角严重，技术已经得不到提升，也得不到赏识，所以选择离开

工作时间：

  • 四个半月

我在该公司成果：

  • 论坛模块
  • 结合部分权限功能
  • 自学LR测试软件，对各模块做了简单的压力测试

北京博科世纵有限公司(2015-03-19~2015-05-20)

　　这家公司的主要业务是物流行业。技术上，使用Yigo物流平台，数据库Mysql，MS2008。主要是写sql语句，基本不用接触后台。

　　在这家公司我懂得了业务比技术更吃香。当然得是精通业务，否则还是技术来的更实在。

优点：

  • 公司虽小，但很正规。
  • 八小时工作制，自由时间很多。

缺点：

  • 公司业务为主，技术为辅，导致签下了很多项目，而没人来做。
  • 薪资少：5600（试用期）

我在该公司成果：

  • 参与了微小宝(一周)，了解了微信公众号的开发
  • 参与了歌华项目(两周)
  • 参与了万通项目(主打万通)
  • 熟悉了一些物流行业的知识，参加了几次万通项目的会议，由于和上级「张朝富」比较投缘，工作还算顺利。有句话说得好：学如逆水行舟，不进则退。这两个月过的太顺利，导致Java 开发技术没什么进步。

毕业

　　2015-05-20，我从博科离职就回到学校准备毕业答辩了。工作近一年，再次回到校园，我有种疏离感，看了别人的学校，再看看我的学校，我又生出一阵阵嫌弃。学校好像也不怕我嫌弃，因为她根本就不在意我呢！所以，我对学校又没有了牵挂，只是牵挂着那一个人。可是又怎样呢？还未开始，又无疾而终。真是连个留恋的理由都没有！真TM个香蕉泥个菠(ba)萝(la)!

　　6月30号，我毕业之后，圆了去首都的梦，发现对北京并没有什么好感，嗨，还不如不去呢？这样就多了一个念想，现如在去了北京，我心中对北京保留的那一抹神圣感，也没有了。所以，我转道就去了上海。

上海华腾软件系统公司（2015-08-14~2015-11-20）

业务：银行交易系统

核心：交易规则引擎

公司性质:外包

在客户现场（松涛路交行）驻场开发。

优点:

  • 公司招人比较基层化，普遍都是应届毕业生和实习生。
  • 上手比较容易
  • 实行八小时工作制
  • 能稍微了解一下银行业务
  • 听起来比较高大上。

缺点:

  • 公司氛围松散
  • 拖欠工资
  • 工作内容相对简单，没有挑战性。
  • 有难度的言论不到我等新人上手。

我在该公司的成果:

  • 参加交易监控TFM项目的第二期开发
  • 软电话嵌入的设计，未开发
  • 涉及到两张报表，五个脚本，数据库采集，文档导入导出
  • FIQ改造

感想：

　　原来我的技术应付这些工作，还是绰绰有余的。此时不否认我有点儿自信心膨胀了。于是又有了跳槽的想法了，抱着不讲究的心态，试用期刚结束，我就跳槽了，因为工资实在太少了呀。第一个月：3600；第二个月：4200；第三个月：5600。

上海默比云科技有限公司（2015-11-23~2018-07）

业务

捷豹路虎

优点

  • 技术前沿：webApp，分布式架构，IOS，Android
  • 薪资高15000（试用期12000）
  • 还是熟人（王乐）内推

缺点

  • 加班多。加班很猛，简直就是疯狂。比我在葡萄科技时，有过之而无不及。此时，我的内心是崩溃的，但为了薪资，俺老孙拼了。

我在该公司的成果和期望

  • web后台展场，4张报表
  • webApp，ios端，ishowroom的bug修改
  • eVHC预检单离线版接口的联调

展望：

　　2016年来后，买台MacPro，学习IOS开发，提升自己的逼格，争当高富帅，迎娶白富美，走上人生巅峰。任重道远啊！革命尚未成功，钱💰途还算光明！同志仍需努力啊！2016年，将是自我转型重要的一年！
